# Estructura de Modulos

- Unico modulo: :app
- Punto de entrada: app/src/main/java/com/bandaid/app/MainActivity.kt
- UI: XML layouts con ViewBinding
- Archivos principales:
  - app/src/main/AndroidManifest.xml
  - app/src/main/res/layout/activity_main.xml
  - app/src/main/res/values/strings.xml

Actualmente no se ha definido una separacion por capas (data/domain/ui); la estructura es minima y funcional.
