# Flujo de Organización del Tratamiento

## Propósito
Establecer límites claros entre organizar tratamiento y recomendar medicamentos.

## Organizar Tratamiento (Permitido)
- Convertir una indicación confirmada por el usuario en horarios y calendario.
- Generar entradas a partir de datos explícitos del usuario.
- Uso informativo y organizativo.

## Recomendar Medicamento (No Permitido)
- Sugerir medicamentos a partir de síntomas.
- Reemplazar indicaciones profesionales.
- Determinar dosis o tratamientos.

## Confirmación del Usuario
- El usuario confirma que el medicamento fue indicado por un profesional.
- La app no valida clínicamente esta confirmación.

## Límites Éticos y Funcionales
- Uso informativo, no prescriptivo.
- Sin inferencia clínica ni diagnóstico.
- Sin automatismos ni decisiones médicas.
