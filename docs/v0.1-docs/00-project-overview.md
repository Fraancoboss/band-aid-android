# Vision General del Proyecto

Band-AId es una aplicacion Android de un solo modulo con un unico punto de entrada en su estado actual (v0.1).

Utiliza layouts XML con ViewBinding y no emplea Jetpack Compose.

No es una biblioteca, no es un proyecto multi-modulo y no incluye componentes de backend.

Estado actual (v0.1): el proyecto compila y se ejecuta localmente cuando el SDK de Android esta configurado.
Requisito vigente: el SDK debe configurarse localmente; no se incluye en el repositorio.
