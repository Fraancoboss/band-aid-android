# Configuracion del Android SDK

El Android SDK es obligatorio y debe configurarse localmente para v0.1.

Se requiere una de estas opciones:
- Definir la variable de entorno ANDROID_HOME con la ruta del SDK.
- Crear local.properties con la clave sdk.dir apuntando a la ruta del SDK.

Hasta que esto se configure, el proyecto no puede compilar.
Con el SDK configurado, v0.1 compila y se ejecuta localmente.
